Mise en page de "Introdution"

Le code HTML

Pour la réalisation de la page d'introduction, nous avons mis une barre de navigation qui permet d'accéder aux différents point clé de la page. Cet élément est commun à toutes les pages réalisés du site, nous avons aussi modifié l'apparence du pointeur lorsque on navigue sur ladite barre.

nous avons par ailleurs aussi implémanté un logo pour l'entreprise Kdou. Nous avons ajouté une barre latérale qui permet d'accéder aux pages des différentes missions du site.

En dessous de la barre de navigation on a le bandeau qui sert de titre pour la page

Concernant la polie d'écriture qui est commun à tous le site c'est "Arial" "Sans-serif"

Le CSS

La barre de navigation est centré avec une distance fixe entre elles, la couleur d'arrière plan est marron foncé, un élément de survol à été inséré, elle ajoute un fond transparent.

Le logo est centré de même

Le bandeau possède une bordure qui permet une légère séparation entre la barre de navigation et celui-ci

Mise en page de "Mission 3"

Le code HTML

Cette mission se caractérise par la création d'un tableau avec différents en-tête

En bas de la page nous avons des images qui ont été insérés ainsi que 3 liens interactif encore en-dessous.

Le CSS

Les 3 liens mentionnées précedemment ont un élément de survol avec fond vert qui arrive en transition. On a aussi de l'ombre.
Ils possèdent chacuns une police et des bordures avec les couleurs concordant avec celles du site.

Concernant les tableaux, Le tableau est séparé en deux blocs gauche et droite, celui de gauche représente les en-têtes et sont mis en gras , ils sont ensuite mis sous la forme de colonnes Il a été implémanté differents éléments de débordement afin de permettre la navigtion du tableau lorsque la fenêtre est rétréci. Même chose si la fenêtre est trop grande
La largeur et longueur des différents colonnes est modifié en fonction des thèmes.
La bordure des différents éléments du tableau sont séparés
